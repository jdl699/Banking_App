# Banking_App
The Banking App is a program that makes the user able to create a bank account, deposit money, and withdraw their money. The user first starts by enter their name into the console and then proceeds to create the bank account with your name.
You can then deposit money into your account with the console promting if you want to desposit,withdraw, or quit/exit. Once depositing money you can see your balance, then you can withdraw what u have dposited and being done showing the transaction and the total amount of money left in the users bank account.
